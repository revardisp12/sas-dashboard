List implementasi

  • 1
    globals.css + tailwind.config
    Ganti semua CSS variable dark ke light. Background utama: #F8F9FC, card: #FFFFFF, sidebar: #FFFFFF, border: #E5E7EB, text: #111827. Hapus neon glow, glassmorphism, dan dark background inline styles.
  • 2
    src/components/Sidebar.tsx
    Ganti background sidebar dari dark ke white. Update warna nav item, section label, brand dropdown, dan active state ke light theme.
  • 3
    src/components/MetricCard.tsx
    Ganti background card dari dark semi-transparent ke white. Border dari rgba ke #E5E7EB. Tambah subtle box-shadow. Hapus glow effect.
  • 4
    src/app/page.tsx + semua platform views
    Ganti semua inline style dark background (#08080F, #0E0E1C, dll) ke light. Update warna teks dan border di chart tooltip, modal, empty state, dsb.
  • 5
    src/lib/types.ts
    Ganti type Timeframe = 7 | 14 | 30 | 90 | 0 menjadi support date range: tambah interface DateRange = { from: string; to: string } atau ubah Timeframe ke Timeframe | DateRange.
  • 6
    src/components/TimeframeSelector.tsx (redesign)
    Redesign komponen: tambah dua date input (From / To) di samping pills. Klik input → tampilkan calendar dropdown dual-month dengan preset shortcuts. Saat user pilih custom range, pills otomatis di-deselect. Saat user klik pill, date range mengikuti.
  • 7
    src/lib/utils.ts
    Update fungsi filterByDays → tambah fungsi baru filterByRange(data, from, to) yang filter berdasarkan date string range. Update semua caller di page.tsx.
Urutan yang disarankan: Kerjain Light Theme dulu (Todo 1–4) karena impactnya besar dan independen. Setelah theme selesai baru Date Range (Todo 5–7) supaya testing lebih mudah.
